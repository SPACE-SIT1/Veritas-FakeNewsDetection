# Dataset

This project uses a fake news dataset obtained from Kaggle.

The full dataset is not included in this repository due to file size and data usage/license considerations.

To run the analysis locally, download the dataset from Kaggle and place the file in this folder as `FakeNews.xlsx`.
Expected columns:
- News_Headline
- Label
- Source
- Link_Of_News

The Python script expects the dataset path to be:

```text
data/FakeNews.xlsx
```
